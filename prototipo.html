🎨 Prototipos en Figma

Diseñamos dos prototipos navegables en Figma que representan las dos interfaces principales del sistema. Estos prototipos fueron validados con usuarios reales antes de iniciar el desarrollo frontend.

🖥️

Panel de Administración

Interfaz web para el administrador del restaurante. Incluye: dashboard con métricas, gestión de productos (CRUD), gestión de categorías, visualización de pedidos en tiempo real y gestión de usuarios.

💡 Decisiones de Diseño
🌑

Tema Oscuro

Los restaurantes suelen tener iluminación tenue. Un tema oscuro reduce la fatiga visual y se ve premium, creando una experiencia agradable para el comensal que consulta el menú en su mesa.

🎯

Mobile-First para Comensal

El comensal accede desde su teléfono vía QR. El diseño se concibió primero para pantallas pequeñas (320px+) y luego se adaptó a escritorio para el panel admin.

👆

Minimal Clicks

El flujo QR → categoría → producto → carrito → pedido requiere el mínimo de interacciones. No hay registro, no hay formularios, no hay fricción.

📊

Dashboard Informativo

El panel admin muestra pedidos recientes, productos más vendidos y estado de mesas de un vistazo, sin necesidad de navegar entre múltiples pantallas.
